一、标签管理

1.1 标签

1.1.1 新建标签

右侧菜单-语义标签,点击进入标签管理界面

1)点击"添加标签"按钮；

2)输入标签名称；

3)添加关键字,可以输入多个关键字,但是要用","隔开；

4)选择所属人,就是给谁添加的,可以多选；

5)单击"保存"按钮；


批量导入

1)点击"导入"按钮,弹出导入框；

2)选择所属人,就是给谁添加的,可以多选；

3)点击"模板下载",下载模板,根据示例,填写上'标签名称' '关键字 '；

4)点击"选择文件"按钮,选择您填好的文件

5)单击"保存"按钮；


1.1.2 编辑标签

只能编辑单个

1)点击"编辑"按钮；

2)修改需要修改的内容；

3)点击"保存"按钮


1.1.3 删除标签

1)点击删除(单个)

2)如果确定要删除,就点击"确定"删除

3)或者(多个), 点击"全选"或单击选择记录前面的复选框,再单击左下角的删除按钮删除多个


1.1.4 标签搜索

1)在搜索框里面输入要搜索的标签名

2)点击搜索按钮
