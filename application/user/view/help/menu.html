菜单管理

这个是管理左侧列表和给角色分配权限用的菜单管理，是权限管理的一环。

1.1 菜单列表

右侧菜单-权限管理-菜单管理,点击进入菜单列表界面


1.2 添加菜单

单击“新增”按钮，进入添加后台菜单页面，填写基本信息后，点击“确定”保存按钮即可。

1)标题 ：这个标题对应的是右侧菜单显示的标题

2)小图标：对应的是右侧菜单显示的小图标

3)排序：字面意思，在列表里面拍第几的意思。

4)链接：就是访问的时候的地址路径，也是代码里面的访问路径。模块/控制器/方法

5)上级菜单：默认是顶级菜单，选择的话可以是二级菜单或顶级菜单，二级菜单上在那个菜单下面的意思

是否隐藏: 隐藏的话，就不在列表里面显示

菜单类型: 必须得选用户，后台是开发的时候用的。

说明: 如果必要，就对这个菜单做一些简单的说明

其他的:分组、仅开发者模式可见都不必填


1.3 编辑菜单

1)点击"编辑"按钮；

2)修改需要修改的内容；

3)点击"保存"按钮


1.4 删除菜单

1)单个：点击删除

2)如果确定要删除,就点击"确定"删除

3)多个：单击选择记录前面的复选框,再单击左上角的删除按钮删除多个，如果确定要删除,就点击"确定"删除


1.5 修改状态

如下图，点击隐藏列的 “是” “否” 关键字，即可修改该菜单的状态。隐藏状态的菜单是看不见的。


1.6 单个修改

如下图，有一些关键字，不用到编辑页面去修改，单击打开弹出框就可以修改了。
