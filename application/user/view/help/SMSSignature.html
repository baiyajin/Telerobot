短信签名管理

1.1 短信签名列表

右侧菜单-短信管理-短信短信,点击进入短信管理界面


1.2 添加短信签名

1)单击“添加签名”按钮，弹出编辑短信签名对话框，填写必要信息后，点击“保存”按钮即可。

1)短信签名：就是发送短信的时候，写在短信内容下面的签名。


1.3 编辑短信签名

字段说明同添加一样

1)点击"编辑"按钮；

2)修改需要修改的内容；

3)点击"保存"按钮


1.4 删除短信签名

1)单个：点击删除，如果确定要删除,就点击"确定"删除

2)多个：点击"全选"或单击选择记录前面的复选框,再单击左下角的删除按钮删除多个，如果确定要删除,就点击"确定"删除


1.5 搜索

1)填写签名

2)点击"搜索"按钮；


1.6 审核内容

为了防止违法的信息传播，加了个审核功能

1) 直接点击表格里面的“审核中”、“通过”关键字即可
