接口配置

配置语音转文字的接口

1.1 接口配置

右侧菜单-系统管理-接口配置,点击进入列表界面


1.2 添加接口

单击“添加接口”按钮，弹出添加接口对话框，填写基本信息后，点击“确定”按钮保存即可。

下面讲几个必要说明的字段

1) app_key：购买接口时，从服务提供商那里获得的app_key。相当于账号吧。

2) app_secret：购买接口时，从服务提供商那里获得的app_secret。相当于密码吧。

3) 接口类型：就是服务提供商吧。


1.3 编辑接口

1)点击"编辑"按钮；

2)修改需要修改的内容；

3)点击"保存"按钮


1.4 删除接口

1)单个：点击删除

2)如果确定要删除,就点击"确定"删除
